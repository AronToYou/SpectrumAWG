stabilize_intensity(which_cam, cam, verbose=False)¶

Balances power across traps.

Given a UC480 camera object (instrumental module) and a number indicating the number of trap objects, applies an iterative image analysis to individual trap adjustment in order to achieve a nearly homogeneous intensity profile across traps.

Parameters

  • which_cam (bool) – True or False selects Pre- or Post- chamber cameras respectively.

  • cam (instrumental.drivers.cameras.uc480) – The camera object opened by instrumental module.
